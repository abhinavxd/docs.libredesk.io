# Libredesk API

Welcome to the Libredesk API documentation. Libredesk is an **open source, self-hosted customer support desk** delivered as a single binary application.

<Note>
  Libredesk is currently in **alpha** stage. Features and APIs may change as we continue development.
</Note>

## Getting Started

The Libredesk API is organized around REST principles and uses standard HTTP response codes. All API endpoints return JSON responses.

## Authentication

Libredesk API supports multiple authentication methods:

- **API Keys**: For server-to-server integration
- **Basic Auth**: For simple username/password authentication  
- **OAuth/OIDC**: For single sign-on integration
