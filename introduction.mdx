# Welcome to Libredesk

Libredesk is an open-source, self-hosted customer support desk — a single binary application designed for simplicity and efficiency.

<img
  src="/images/hero.png"
  alt="Libredesk Dashboard"
  className="rounded-lg border"

## What is Libredesk?

Libredesk is a modern helpdesk solution that gives you complete control over your customer support infrastructure. Built as a single binary with a Vue.js frontend, it's designed to be easy to deploy, maintain, and customize.

## Why Libredesk?

- **Privacy First**: Your data stays on your servers
- **Cost Effective**: No per-agent pricing or usage limits
- **Developer Friendly**: Open-source with a clean API
- **Production Ready**: Battle-tested with enterprise features

## Tech Stack

Libredesk is built with modern, reliable technologies:

- **Backend**: Go (Golang)
- **Frontend**: Vue 3 with Shadcn UI
- **Database**: PostgreSQL (≥13)
- **Cache**: Redis
- **Real-time**: WebSockets
